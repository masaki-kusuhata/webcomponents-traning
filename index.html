JavaScript 復習

GitHub からのソースコードの取得方法を覚える

JavaScritp ではありませんが、教材などが GitHub 上にあることはよくあります。GitHub からソースコードを持ってくる方法を覚えておきましょう!

無名関数の即時実行

(function () {
  // ここに処理を書く
})();

JavaScript のコードは無名関数の即時実行で囲むのは癖をつけましょう! プロダクトコードだと必須です! そんなのどうでもいいじゃんと思っている人はなぜなのか調べてみましょう!

strict モード

(function () {
  'use strict';

  // ここに処理を書く
})();

JavaScript のコードを書くときは strict モードにするのを忘れないようにしましょう! JavaScript エンジンの挙動が変わるため、あとからの変更はほぼ不可能です。これもどうでもいいじゃんと思っている人はなぜなのか調べてみましょう!

querySelector 関数など

querySelector 関数などa>の DOM 操作のための基礎的な方法はひと通り覚えましょう! Mozillla Developer Network さまさまです。ハンズオンの中で使った関数はこまめに調べましょう!

Emmet とか

ハンズオンではゆっくりコードを書く時間がなくなることもたくさん。Emmet などのスニペットツールや入力補完を使うことで短い時間でも書けるようになります。その分心に余裕が生まれますので、講師の話を聞く方に脳みそを使えます。

コード検査

JavaScript のような LL は簡単に書き始められる反面、スペルミスなどのミスを検出しづらい傾向があります。ちょっとしたミスでも気が付かないと時間を失うため、ハンズオンでは慌てることになります。

JSHint + JSCS や ESLint などの検査ツール、もしくはエディターの持っている検査機能をリアルタイムに使うことで、つまらないミスを防ぎ、ハンズオンで遅れることを防ぐことができます。
